# Market Lens

Personal investment intelligence dashboard for family use.

## What it does
- Ingests news from across the web (RSS + a few public feeds)
- Deduplicates/clusters stories into "events"
- Uses an LLM analysis pipeline to generate structured **investment insight objects**
- Serves a simple dashboard UI + a weekly digest email
